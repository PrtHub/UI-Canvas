## Usage

```tsx
import {
  PainterlySelect,
  PainterlySelectContent,
  PainterlySelectGroup,
  PainterlySelectItem,
  PainterlySelectLabel,
  PainterlySelectTrigger,
  PainterlySelectValue,
} from "@/components/ui/painterly-select";

export default function PainterlySelectExample() {
  return (
    <PainterlySelect>
      <PainterlySelectTrigger className="w-[180px]">
        <PainterlySelectValue placeholder="Select a color" />
      </PainterlySelectTrigger>
      <PainterlySelectContent>
        <PainterlySelectGroup>
          <PainterlySelectLabel>Colors</PainterlySelectLabel>
          <PainterlySelectItem value="red">Crimson Red</PainterlySelectItem>
          <PainterlySelectItem value="blue">Ocean Blue</PainterlySelectItem>
          <PainterlySelectItem value="yellow">
            Sunflower Yellow
          </PainterlySelectItem>
        </PainterlySelectGroup>
      </PainterlySelectContent>
    </PainterlySelect>
  );
}
```

## API Reference

### PainterlySelect

The root component that contains all the parts of a select.

```tsx
<PainterlySelect>
  <PainterlySelectTrigger>
    <PainterlySelectValue />
  </PainterlySelectTrigger>
  <PainterlySelectContent>
    <PainterlySelectItem>
      {/* ... */}
    </PainterlySelectItem>
  </PainterlySelectContent>
</PainterlySelect>
```

### Properties

- `defaultValue` - The value of the select when initially rendered
- `value` - The controlled value of the select
- `onValueChange` - Event handler called when the value changes
- `disabled` - When true, prevents the user from interacting with the select
- `name` - The name of the select when used in a form
- `required` - When true, indicates that the user must select a value before the form can be submitted

### PainterlySelectTrigger

The button that toggles the select. By default, the `PainterlySelectValue` will be rendered inside the trigger.

### PainterlySelectContent

The component that contains the select items. By default, it renders in a portal and has artistic animations.

### PainterlySelectItem

The component that contains the select items. Has artistic hover and focus states.
