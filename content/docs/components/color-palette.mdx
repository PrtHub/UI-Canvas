## Use Cases

The Color Palette component is designed to enhance your application's visual design system in various scenarios:

1. **Theme Customization**

   - Use in theme settings or customization panels where users can select color schemes
   - Perfect for applications with user-customizable interfaces
   - Ideal for design tools or creative applications

2. **Design System Documentation**

   - Display your application's color schemes in documentation
   - Showcase different theme variations
   - Present brand color guidelines

3. **Creative Applications**

   - Art or design applications where users need to select from predefined palettes
   - Digital painting tools where different artistic styles are available
   - Educational platforms teaching color theory or artistic techniques

4. **Mood Boards**
   - Create dynamic mood boards with different artistic styles
   - Present color combinations for design projects
   - Showcase different aesthetic directions

### Example Implementation

```tsx
import { ColorPalette } from "@/components/ui-canvas/color-palette";

// Theme switcher example
export function ThemeCustomizer() {
  return (
    <div className="space-y-4">
      <h3>Choose Your Theme Style</h3>
      <ColorPalette
        variant="watercolor"
        onChange={(colors) => updateTheme(colors)}
      />
    </div>
  );
}
```

## Props

### ColorPalette

| Prop name | Type                                                              | Default      | Description                       |
| --------- | ----------------------------------------------------------------- | ------------ | --------------------------------- |
| variant   | "watercolor" \| "oilpaint" \| "charcoal" \| "pastel" \| "acrylic" | "watercolor" | The artistic style of the palette |
| showNames | boolean                                                           | true         | Show color group names            |
| className | string                                                            | -            | Additional CSS classes            |

## Features

The Color Palette component provides:

- Five distinct artistic styles: Watercolor, Oil Paint, Charcoal, Pastel, and Acrylic
- Three color groups per palette: Primary, Secondary, and Accent
- Three shades per color group
- Hover effects to display color codes
- Responsive and mobile-friendly design
- Customizable appearance through className prop

## Color Schemes

Each palette variant includes carefully curated colors that reflect the characteristics of their respective artistic mediums:

- **Watercolor**: Soft, flowing blues and purples with aqua accents
- **Oil Paint**: Rich, warm oranges and reds with deep undertones
- **Charcoal**: Monochromatic grays with subtle variations
- **Pastel**: Light, airy pinks and peaches with sunny accents
- **Acrylic**: Vibrant greens and teals with bright highlights

## Accessibility

The Color Palette component follows accessibility best practices:

- Color codes are displayed on hover for accurate color identification
- Sufficient contrast between text and background colors
- Interactive elements are keyboard accessible
- Color information is conveyed through both visual and text means
