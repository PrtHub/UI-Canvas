## Examples

### Basic Usage

```tsx
export function BasicToast() {
  return (
    <PaletteToast message="A splash of information has arrived!" type="info" />
  );
}
```

### Custom Duration

```tsx
export function TimedToast() {
  return (
    <PaletteToast
      message="Your masterpiece has been created!"
      type="success"
      duration={3000}
    />
  );
}
```

## Props

### PaletteToast

| Prop      | Type                                        | Default | Description                                   |
| --------- | ------------------------------------------- | ------- | --------------------------------------------- |
| message   | string                                      | -       | The message to display in the toast           |
| type      | "info" \| "success" \| "warning" \| "error" | "info"  | The type of toast, affects colors and styling |
| duration  | number                                      | 5000    | Duration in ms before toast auto-dismisses    |
| onClose   | () => void                                  | -       | Callback function when toast is dismissed     |
| className | string                                      | -       | Additional CSS classes for styling            |
