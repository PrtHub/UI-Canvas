### Variant Usage Guide

- **Default**: Clean, minimal style for general use
- **Watercolor**: Soft, flowing effects for gentle emphasis
- **Oil**: Bold, textured look for strong visual impact
- **Charcoal**: Dramatic, sketchy style for artistic contrast

### Example Implementation

```tsx
import { CanvasButton } from "@/components/ui-canvas/canvas-button";

// Portfolio contact button
export function ContactSection() {
  return (
    <div className="space-y-4">
      <h3>Get in Touch</h3>
      <CanvasButton variant="watercolor" onClick={() => openContactForm()}>
        Start a Project
      </CanvasButton>
    </div>
  );
}

// Art gallery navigation
export function GalleryNav() {
  return (
    <div className="flex gap-4">
      <CanvasButton variant="oil">Previous Artwork</CanvasButton>
      <CanvasButton variant="oil">Next Artwork</CanvasButton>
    </div>
  );
}
```

## Props

### CanvasButton

| Prop name | Type                                             | Default   | Description                 |
| --------- | ------------------------------------------------ | --------- | --------------------------- |
| variant   | "default" \| "watercolor" \| "oil" \| "charcoal" | "default" | The button's artistic style |
| size      | "sm" \| "default" \| "lg"                        | "default" | The button's size           |
| effect    | "none" \| "splash" \| "brush"                    | "none"    | The interactive effect      |
| asChild   | boolean                                          | false     | Merge with child element    |
| className | string                                           | -         | Additional CSS classes      |

## Features

The Canvas Button component brings artistic flair to your UI with:

- Three distinct artistic styles: Watercolor, Oil Paint, and Charcoal
- Interactive effects: Paint splash on click and brush stroke on hover
- Multiple sizes and fully customizable appearance
- Accessible and keyboard-navigable
- Responsive and mobile-friendly

## Accessibility

The Canvas Button component follows WCAG guidelines:

- Maintains color contrast ratios
- Supports keyboard navigation
- Includes focus indicators
- Works with screen readers
